- Concept learning as search
- Hypothesis representation
- Find-S Algorithm
- Version Space
- Candidate Elimination Algorithm
- Inductive Bias

### Decision Trees

- Decision tree representation
- Hypothesis space search
- Information Gain
- ID3 Algorithm
- C4.5 Algorithm
- Overfitting & Pruning
- Inductive Bias

### Neural Networks

- Perceptron model
- Activation functions
- Multilayer Neural Networks
- Backpropagation algorithm
- ANN representation

## 📕 Module 03 — Hypothesis Evaluation & Bayesian Learning

### Statistical Evaluation

- Estimating hypothesis accuracy
- Sampling theory
- Confidence intervals
- Comparing algorithms
- Cross-validation
- Error difference estimation

### Bayesian Learning

- Bayes Theorem
- Maximum A Posteriori (MAP)
- Maximum Likelihood Estimation (MLE)
- Least Squares Error
- Minimum Description Length (MDL)

## 📙 Module 04 — Bayesian & Instance-Based Learning

### Bayesian Methods

- Naive Bayes Classifier
- Bayesian Belief Networks
- Conditional independence
- Expectation-Maximization (EM)

### Instance-Based Learning

- k-Nearest Neighbour (k-NN)
- Distance metrics
- Curse of dimensionality
- Locally Weighted Regression

### Kernel Methods

- Radial Basis Function Networks
- Case-Based Learning

## 📓 Module 05 — Reinforcement & Analytical Learning

### Reinforcement Learning

- Learning from reward
- Markov Decision Processes (MDPs)
- Q-Learning Algorithm
- Exploration vs Exploitation
- Non-deterministic rewards

### Analytical Learning

- Explanation-Based Learning (EBL)
- Prolog-EBG
- Learning with domain theories
- Knowledge transfer
